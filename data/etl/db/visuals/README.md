 Telecom Sales Data Pipeline

Goal: Load telecom_sales.csv, clean it, store in SQLite, run simple analysis, and plot a chart.

## Structure
- data/: telecom_sales.csv (raw), cleaned_telecom_sales.csv (cleaned)
- etl/: Python scripts (inspect, clean, load, query, plot)
- db/: SQLite database (telecom_sales.db)
- visuals/: exported charts

## Steps
1. Ensure data/telecom_sales.csv exists.
2. Run inspect → clean → load → query → plot (scripts added later).
